# CSS position

## Position 유형
1. static
2. relative 상대적 --- 본인 공간 유지 후 이
3. absolute 절대적 ---- 본인 공간 없앰
4. fixed 고정
5. sticky 끈적끈

## z-index

-static이 아닌 요소에만 적용됨

### Position의 목적
